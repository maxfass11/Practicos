Web Storage

Web Storage es un concepto que se refiere a una API proporcionada por los navegadores web para permitir que las aplicaciones web almacenen datos de manera persistente en el lado del cliente. Hay dos tipos principales de Web Storage:

  • localStorage: El almacenamiento local (localStorage) permite a las aplicaciones web almacenar datos de manera persistente en el navegador web es decir que permanecen incluso despu√©s de cerrar y volver a abrir el navegador.
  • sessionStorage: Los datos almacenados tienen una duracion limitada a la sesion del usuario, es decir que se perderan si se cierra la ventana del navegador.
