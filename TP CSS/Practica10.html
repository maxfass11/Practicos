Elemento <iframe>

El elemento <iframe> (Inline Frame) en HTML se utiliza para incrustar otro documento HTML dentro de la página actual. Esto permite mostrar contenido de otra fuente o página web dentro de la página principal.

Algunos casos comunes de uso del elemento <iframe> son la inclusion de contenido externo como videos de Youtube, visualizacion de documentos PDF, contenido embebido de otros sitios que lo permiten como X.com, etc.
